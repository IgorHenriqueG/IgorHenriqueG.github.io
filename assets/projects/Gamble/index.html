This game is a simple version of the classic game "Roulette".

It was created as a project for learning HTML, CSS and JavaScript.

The first spin will be a win if the bet is $40 or less.

Each symbol will be multiplied by his value, like the emerald, if you win it wil be your bet * the symbol value.
